# TRETMAX Website

A simple multi-page static website for TRETMAX.

## Usage

Open `index.html` in your web browser to view the home page. Navigate using the menu to other pages.

## Files

- `index.html`: Home page
- `about.html`: About page
- `services.html`: Services page
- `contact.html`: Contact page
- `styles.css`: CSS styles
- `script.js`: JavaScript (currently minimal)

## Troubleshooting

If the website doesn't load properly, ensure all files are in the same directory.
